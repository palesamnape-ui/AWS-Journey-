# AWS Lab: Selecting Data from a Database (SQL SELECT Queries) 270

## Overview

Welcome to this project repository! This lab documents my completion of an **AWS database fundamentals lab focused on querying relational data using SQL**.

In this exercise, I connected to a preconfigured database environment hosted on an **Amazon EC2 Command Host** and queried a relational database named **`world`**. Using the `SELECT` statement, comparison operators, and aggregate functions, I retrieved and filtered data across multiple tables.

---

## The Lab Objectives

After completing this lab, I was able to:

* Connect to a database instance from an EC2 Command Host
* Use the `SELECT` statement to retrieve data
* Apply the `COUNT()` function
* Filter results using `WHERE`, comparison operators (`<`, `>`, `=`), and logical operators (`AND`)
* Sort query results using `ORDER BY`
